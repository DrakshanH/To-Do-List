Simple Task Manager
This is a simple Python program designed to help you manage your tasks. It allows you to add tasks, delete them, move them between different categories (To-Do, In-Progress, Done), and view a list of all your tasks.


Features
Add new tasks.
Delete tasks from any category (To-Do, In-Progress, Done).
Move tasks between categories.
View a list of all tasks categorized.
Exit the program.


Usage
1. Run the program by saving it as a Python file (e.g., task_manager.py) and executing it from the command line:
      python task_manager.py

2. The program will present you with a menu offering various options. Choose the desired option by entering the corresponding number:
* Add a new task
* Delete a task
* Shift tasks
* List tasks
* Exit

3.Follow the on-screen prompts to complete your chosen action.

Additional Notes
The program currently stores tasks in memory only. Once you exit the program, all tasks will be lost.
This is a basic example and can be extended with additional functionalities like setting deadlines, priorities, etc.
